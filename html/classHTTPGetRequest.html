void HTTPGetRequest::ReadData ( )
private

Helper function for void sendRequest();.

The function sendRequest() is designed to read only the header of the HTTP response from the server. The contents of the response (the image file or the text for a text file or an html text for an html file) should be read by this function, porion by portion.
