Простое клиент-серверное приложение, работающее по http протоколу. С сервера на клиент передаются текстовые, html и графические файлы.
Сначала необходимо запустить сервер. Находясь в корневой директории проекта, перейти в папку server/ и выполнить команды:
$cmake -S . -B build/
$cmake --build build/
Далее запустить сервер:
$./build/http-server <номер порта> (например, $./build/http-server 10000)
Далее необходимо запустить клиент. Для этого, находясь в корневой директории проекта, перейти в папку client/ и выполнить команды:
$cmake -S . -B build/
$cmake --build build/
Далее запустить клиент:
$./build/http-client <ip-адрес сервера> <номер порта сервера> (например, $./build/http-client 127.0.0.1 10000)
Здесь для локальной машины ip-адрес может вводиться в виде "127.0.0.1" либо "localhost".
Появляется пустая строка, куда необходимо ввести URI запрашиваемого с сервера ресурса.
Все файлы ресурсов хранятся в папке server/data/. В подпапке server/data/images хранятся изображения, а в server/data/text - текстовые файлы.
Например, вводим:
$/data/images/1.jpg
Сервер должен передать клиенту этот файл и сохранить его в папке клиента (client/) в выходной файл received.jpg (расширение такое же).
Далее, этот файл можно открыть, например, с помощью okular.
Если нужно передать текстовый файл (таблица из 2 колонок), то вводим:
$/data/text/data.txt

Клиент способен осуществлять много вызовов за одну сессию. После получения ответа с сервера он может опять вводить новый запрос. Если файл файл существует, то в директории client/ появится файл received с таким же расширением, как и запрашиваемый.

Чтобы завершить клиент, это можно сделать при помощи Ctrl+C либо любой из команд (регистр не имеет значения):
EXIT
QUIT
X
Q

Сервер можно остановить нажатием Ctrl+C в терминале, где он открыт, либо отправкой с клиента одной из команд (регистр букв не имеет значения):
SERVER SHUTDOWN
SERVER EXIT
SERVER STOP
SERVER FINISH
